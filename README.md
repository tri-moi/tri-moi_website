# Tri&Moi : site web

## Installer le projet

Apres avoir cloné le projet, l'ouvrir dans un terminal puis installer les dépendances aves la commande `yarn` ou `yarn install`.

## Lancer le serveur

lancer `ng serve` dans le terminal de votre projet.
